## Pre-requisites
#### Install Angular13
```
npm install -g @angular/cli
```

### Angular Material for Frontend
```
ng add @angular/material
```

### Install Bootstrap to assist in UI development
```
npm install bootstrap
```


## Run the web application

### Backend

```
cd backend/src
```

```
go run *.go
```

### Frontend

```
cd frontend/web-app/
```
```
ng serve
```

Open browser port

```
localhost:4200
```
